# http-implementation
基于C Socket的http请求底层实现
目前拥有GET, POST两种请求方式

# Using
$> client <ip> <port>
不附加参数默认为localhost:8080
$> enter GET: <method>
$> enter /: <index>
$> enter C/Socket: <User-Agent>
$> enter 127.0.0.1: <host>
$> enter text/html: <Content-Type>
$> enter 0-: <Range>
$> enter Keep-Alive: <Connection>
